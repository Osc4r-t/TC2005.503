# Frontend

Proyecto Node.js con Express para servir archivos estaticos desde la carpeta `public`.

## Requisitos

- Node.js
- npm

## Instalacion

```bash
npm install
```

## Uso

```bash
npm start
```

El servidor se levanta en:

```text
http://localhost:8080
```

## Estructura

- `index.js`: inicia el servidor Express
- `public/`: contiene los archivos HTML, CSS, JS e imagenes del frontend
